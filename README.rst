Instatools
==========

|Supported Python Versions| |License|

Instagram scraping tools.

::

    make dev
    . venv/bin/activate
    instaconfig -h
    instafeed -h

Roadmap
-------

- Download media files.
- ``instascrape`` for full timeline scraping (using the https://www.instagram.com/graphql/query/ endpoint).
